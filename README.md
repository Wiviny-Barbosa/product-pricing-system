# Sistema de Precificação de Produtos (Java)

Projeto desenvolvido para praticar Programação Orientada a Objetos com foco em herança e polimorfismo.

## Sobre
O sistema cadastra produtos comuns, usados e importados, calculando o preço final de cada um de forma polimórfica.

Produtos importados possuem taxa adicional, e produtos usados exibem a data de fabricação.

## Funcionalidades
- Cadastro de produtos
- Cálculo do preço final
- Exibição de etiquetas de preço

## Conceitos aplicados
- Herança
- Polimorfismo
- Sobrescrita de métodos
- Listas de objetos
- Manipulação de datas (LocalDate)

## Tecnologias
- Java
